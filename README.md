# 🤖 AI Chatbot with GUI using Python

## 📌 Project Overview

This project is a **GUI-based AI Chatbot** developed using **Python** and **Tkinter**.
The chatbot interacts with users through a graphical interface, responds to basic queries, and performs simple tasks such as displaying the current time and date.

It also stores conversation history automatically in a text file, making the application more practical and professional.

This project was created as part of an **Artificial Intelligence Internship** to demonstrate basic Natural Language Processing concepts and GUI development using Python.

---

## ✨ Features

* User-friendly **Graphical User Interface (GUI)**
* Responds to user messages
* Displays **current time and date**
* Automatically saves chat history
* Clear chat functionality
* Simple and interactive chatbot experience

---

## 🛠️ Technologies Used
